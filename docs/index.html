【説明】

  1. 「接続」で接続器を選択する（※Chrome、Edge以外のブラウザでは接続できません）
  2. 「感度送信」でデフォルトの感度を書き込むか、「感度要求」で感度を読み込む
  3. 感度を調整する
  4. 「感度送信」で接続器に感度を送信する
  5. 必要に応じて「保存」を行う（接続器のメモリに保存されます）

感度

それぞれの感度です。値は閾値になっていて、センサーからの電圧が一定を超えた場合にキー入力を行います。値が低いほど感度が高くなります。多重入力が発生する箇所は感度を高くしましょう。

Adelay
キーが入力されてから次にそのキーが入力を受け付けるまでの時間です。大きいほど多重入力が起きにくくなります。叩いた時にその部分のみで多重入力が発生するなら値を上げてください。
標準: 1 ms

Bdelay
面のキーが入力されてから次の面の入力を受け付けるまでの時間です。大きすぎると面の連打が伸びなくなります。
標準: 12 ms

Cdelay
BdelayのKDelay版（縁バージョン）です。
標準: 30 ms

Ddelay
面が入力されてから縁の入力を制限する時間（兼その逆）。面を叩いたときに縁が多重する場合などに値を上げてください。
標準: 32 ms

Hdelay
公式アプリや一部のシミュレーターで必要な入力制限です。「メモ帳ではキー入力できるがシミュレーターでは入力できない」場合に設定してください。
標準: 20 or 0 ms

【感度調整のやり方】

基本的には1〜3を繰り返し、できるだけ高い感度・低いdelayを目指してください。

  1. 感度設定：デフォルトの感度に設定してから、バチで叩き無反応が起きない値を探します。
  2. delay調整：多重入力が起きないようにそれぞれの値を上げます。目安として値が35以上になるとロール処理が失敗しやすくなります。
  3. 感度を調整：delay調整でカバーできない多重入力が発生した場合は感度を下げてください。
