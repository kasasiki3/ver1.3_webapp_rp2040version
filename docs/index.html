使い方
  1. 「接続」で接続器を選択する ※ Chrome / Edge 以外では接続できません
  2. 「感度送信」でデフォルト感度を書き込むか、「感度要求」で現在の感度を読み込む
  3. スライダーで感度・ディレイを調整する
  4. 「感度送信」で接続器に感度を送信する
  5. 必要に応じて「保存」を行う 接続器のフラッシュメモリに書き込まれます

右縁

各センサーの閾値です。センサーの電圧がこの値を超えるとキー入力が発生します。値が低いほど感度が高くなります。多重入力が発生する箇所は感度を上げて（値を大きく）ください。

Adelay 標準: 1 ms

キー入力後、同じキーが再び受け付けるまでの待機時間。叩いた箇所のみで多重入力が起きる場合に値を上げてください。

Bdelay 標準: 12 ms

面キー入力後、次の面入力を受け付けるまでの時間。大きすぎると面の連打が鈍くなります。

Cdelay 標準: 30 ms

Bdelay の縁バージョンです。説明は Bdelay と同様です。

Ddelay 標準: 32 ms

面入力後に縁入力を制限する時間（逆も同様）。面を叩いたときに縁が多重する場合や、その逆の場合に値を上げてください。

Hdelay 標準: 20 or 0 ms

公式アプリや一部シミュレーター向けの入力制限。「メモ帳では入力できるがシミュレーターでは入力できない」場合に設定してください。

感度調整のやり方

基本的に 1 → 3 を繰り返し、できるだけ高い感度・低いディレイを目指してください。

  1. 感度設定 デフォルト値から始め、バチで叩いて無反応が起きない最小値を探します。
  2. ディレイ調整 多重入力が起きないよう各ディレイを上げます。目安: 35 以上になるとロール処理が失敗しやすくなります。
  3. 感度の微調整 ディレイで解決できない多重入力が残る場合は感度を下げます（値を大きくする）。
